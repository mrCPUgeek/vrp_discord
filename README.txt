vrp_discord by mr_cpu_geek Version 1.0

Adds discord to your chat

Commands:
/ooc:  Will show in chat as "OOC|UserID"
/me: Will show in chat as "ME|CharacterName"
/tweet or /twitter:  Will show in chat as "Twitter|CharacterName" and will send to discord
/911: Will show in chat as "911|CharacterName" and will send to discord
/news: Will show in chat as "News|Charactername" and will send to discord
/ad: Will show in chat as "AD|CharacterName" and will send to discord
/do: Will show in chat as "DO|CharacterName"

Also when the server is started, a nottice will be sent to discord.
